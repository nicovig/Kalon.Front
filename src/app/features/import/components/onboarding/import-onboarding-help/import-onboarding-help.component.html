Besoin de précisions ?

Après le dépôt, vous reliez chaque colonne à Kalon ; l’aperçu se met à jour tout seul.

  • À gauche, les titres de colonnes de votre fichier (souvent la première ligne).
  • À droite, le champ Kalon correspondant : nom, email, téléphone, etc.
  • L’aperçu se met à jour tout seul pour vérifier que tout est au bon endroit.
Conseils utiles
  • Une colonne inutile ? Choisissez « Ignorer » dans la liste.
  • Champs obligatoires (donateurs) :
    • Particuliers : Email + ( Nom ou Prénom ).
    • Entreprises : Email + Nom entreprise + SIRET + ( Nom ou Prénom du contact ).
  • Champs obligatoires (dons) : Date + Montant + lien avec Kalon (email du donateur ou nom/prénom).
  • Si vous ne savez pas quoi choisir, laissez notre détection automatique ou testez une autre option et regardez l’aperçu.
  • Plusieurs colonnes pour la même chose (par exemple deux colonnes « email ») ? Un message vous le signale : gardez une seule colonne active ou changez les choix pour éviter les doublons.
